# Daily Checklist with Focus Timer

Stay organized and boost your productivity with the **Daily Checklist** application! This program helps you manage your daily tasks efficiently while incorporating a built-in **Focus Timer** to keep you on track.

## Key Features

- **Task Management**: Add, rename, and remove tasks for your morning and evening routines.
- **Customizable Scheduling**: Assign tasks to specific days or set them as daily or weekday tasks.
- **Focus Timer**: Start a timer for any task to stay focused and track your progress.
- **Interactive UI**: Easy-to-use interface with context menus and tabbed navigation.
- **Visual Feedback**: Track completed time chunks with a dynamic circle panel.
- **Modern About Dialog**: View application information in a sleek, modern dialog.
- **Robust Error Handling**: Comprehensive error messages for file operations and data issues.

Perfect for anyone looking to streamline their daily routines and stay focused on what matters most!



## Use Cases

### 1. Add a New Task

- Navigate to the "Add Task" tab.
- Enter the task name in the text field.
- Select the time of day (Morning or Evening).
- Choose the frequency (Daily, Weekday, or a specific day).
- Click "Add Task" to save it to your checklist.

### 2. Start a Focus Timer for a Task

- Go to the "Checklist" tab.
- Right-click on a task and select "Start Focus Timer."
- Click "OK" to start the timer.

### 3. Rename or Remove a Task

- Right-click on a task in the checklist.
- Select "Rename" to edit the task name or "Remove" to delete it.
- Confirm your changes in the dialog box.

### 4. Track Progress with Circles

- While the timer runs, observe the circle panel to track completed time chunks.
- Each circle represents a 5-minute interval.
