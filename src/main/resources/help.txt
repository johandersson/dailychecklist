			<li>You can add subtasks to any daily or custom task. Subtasks appear directly below their parent and are indented for clarity.</li>
			<li>To create a subtask, use the "Add Task" dialog and select a parent task (if supported in the UI), or right-click a task and choose "Add Subtask" (if available).</li>
			<li>When all subtasks are marked as done, the parent task is automatically marked as done.</li>
			<li>When you mark a parent task as done, all its subtasks are also marked as done. Unmarking a parent will unmark all subtasks.</li>
			<li>Subtasks are supported in both daily and custom checklists.</li>

	<h1>Daily Checklist Help</h1>

	<p>Daily Checklist is a focused app for managing daily routines (not general todo history). It records whether a task was completed each day but resets status daily. This app is designed for quick, repeatable daily checklists rather than long‑lived task tracking.</p>

	<h2>Getting Started</h2>
	<ol>
		<li>Launch the app: run <code>java -jar dailychecklist.jar</code> or double-click the JAR file.</li>
		<li>Main interface: a split layout shows Daily tasks (left) and Custom checklists (right).</li>
	</ol>

	<h2>Notes</h2>
	<ul>
		<li>Icons and visuals have been improved for smoother performance and a more consistent look.</li>
		<li>When you move a task between checklists the app will switch to the target checklist and show the moved task; use drag-and-drop to move tasks.</li>
	</ul>

	<h2>Managing Daily Tasks</h2>
	<h3>Adding a new daily task</h3>
	<ul>
		<li>Click the "Add Task" button or press <b>Ctrl+A</b>.</li>
		<li>Enter a name (for example, "Morning Exercise").</li>
		<li>Choose Morning or Evening and frequency (Daily / Weekday / Specific Day).</li>
		<li>Click "Add Task" — it appears in the left panel under the chosen time.</li>
	</ul>

	<h3>Viewing and editing daily tasks</h3>
	<ul>
		<li>The left panel groups tasks by Morning and Evening.</li>
		<li>Use the "Show all weekday tasks" option to include weekday-only items.</li>
		<li>Right-click a task to rename or remove it, or start the focus timer.</li>
	</ul>

	<h3>Completing tasks</h3>
	<ul>
		<li>Click a task's checkbox to toggle completion.</li>
		<li>Click or double-click task text to select or toggle.</li>
		<li>Multi-select with <b>Ctrl+click</b>; range-select with <b>Shift+click</b>.</li>
		<li>Tasks reset daily; completion does not carry over.</li>
	</ul>

	<h2>Custom Checklists</h2>
	<h3>Create a checklist</h3>
	<p>In the right panel, enter a name and click "Create Checklist". The name cannot be empty.</p>

	<h3>Add tasks to a checklist</h3>
	<p>Select a checklist, enter tasks (one per line) in the Add Tasks area, and click "Add Tasks". Empty lines are skipped. Tasks appear in the selected checklist.</p>

	<h3>Manage checklists</h3>
	<ul>
		<li>Right-click a checklist to Rename, Delete, Add Reminder or View/Edit Reminders.</li>
		<li>Checklists with reminders display a small clock indicator and a textual status in the UI (e.g. "Due soon", "Overdue").</li>
	</ul>

	<h3>Editing custom tasks</h3>
	<ul>
		<li>Toggle completion with the checkbox; double-click to rename; right-click for more options.</li>
		<li>Drag and drop tasks to reorder or move between checklists.</li>
	</ul>

	<h2>Searching Tasks</h2>
	<p>Press <b>Ctrl+F</b> or use the Search menu to open the search dialog. Type to filter tasks across all checklists in real time (case-insensitive). Select a result and click "Go to Selected Task" to jump to it in the UI.</p>

	<h2>Reminders</h2>
	<h3>Add and manage</h3>
	<ul>
		<li>Right-click a checklist → <i>Add Reminder</i>. Set date and time.</li>
		<li>Right-click → <i>View/Edit Reminders</i> to modify or delete reminders.</li>
	</ul>

	<h3>Notification states</h3>
	<ul>
		<li><b>Future</b>: scheduled more than 60 minutes away.</li>
		<li><b>Due soon</b>: within the next 60 minutes.</li>
		<li><b>Overdue</b>: scheduled time has passed.</li>
		<li><b>Very overdue</b>: UI indicates critical overdue reminders.</li>
	</ul>

	<h2>Focus Timer</h2>
	<p>Right-click a task → <i>Start Focus Timer</i>. Enter minutes (e.g., 25) and start. During the timer a countdown window appears; controls: Space = Pause/Resume, Enter = Start (when paused).</p>

	<h2>Settings & Troubleshooting</h2>
	<ul>
		<li>Access settings via the menu bar to customize defaults (timer length, etc.).</li>
		<li>If the app won't start: verify Java with <code>java -version</code>.</li>
		<li>Data loading issues: see File → Restore from Backup and backups in your user folder.</li>
		<li>Tasks not saving: check file permissions. Data files are stored under your user directory.</li>
	</ul>

	<h2>Data Backup and Recovery</h2>
	<ul>
		<li>Backups are stored in <code>dailychecklist-backups</code> in your user directory.</li>
		<li>The app makes automatic backups on startup and periodically.</li>
		<li>To recover, use File → Restore from Backup (the app backs up your live data first).</li>
	</ul>

	<h2>Keyboard Shortcuts</h2>
	<ul>
		<li>Add Task: <b>Ctrl+A</b></li>
		<li>Search Tasks: <b>Ctrl+F</b></li>
		<li>Refresh Tasks: <b>F5</b></li>
		<li>Multi-select: <b>Ctrl+click</b></li>
		<li>Range-select: <b>Shift+click</b></li>
		<li>Delete selected tasks: <b>Delete</b></li>
		<li>Timer pause/resume: <b>Space</b></li>
	</ul>

	<h2>Reminder Status Guide</h2>
	<ul>
		<li><b>Overdue</b> — the scheduled time has passed.</li>
		<li><b>Due soon</b> — scheduled within the next 60 minutes.</li>
		<li><b>Future</b> — scheduled more than 60 minutes away.</li>
	</ul>

	<p>For more details see the <a href="https://github.com/yourrepo/dailychecklist">project repository</a> or the README.md.</p>

	<p>Enjoy staying productive!</p>
